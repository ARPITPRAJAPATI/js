# JavaScript Learning Journal 📘

This repo is my personal space for learning and practicing JavaScript — from the basics all the way to DOM manipulation, OOP, and small real-world projects. Everything here is organized topic by topic, so I (and anyone else browsing) can quickly find what was covered and revise it later.

## 📂 Folder Structure

| Folder | What's inside |
|---|---|
| `basics/` | Core fundamentals — variables, data types, type conversion, and working with strings, numbers, and dates |
| `basics2/` | Arrays and objects — creating, accessing, and manipulating them |
| `basics3/` | Functions — arrow functions, IIFEs (immediately invoked function expressions), and scope |
| `control_flow/` | Conditionals, switch statements, and truthy/falsy values |
| `iterations/` | Loops and iteration patterns (`for`, `while`, etc.) |
| `oops/` | Object-Oriented Programming — classes, prototypes, inheritance, getters/setters, `call`/`bind` |
| `dom/` | DOM (Document Object Manipulation) basics — selecting and updating HTML elements with JS |
| `events/` | Handling browser events (clicks, key presses, etc.) |
| `advance_one/` | Slightly advanced topics — Promises and making API requests |
| `projects/` | Small hands-on projects that combine everything learned above |

## 🚀 Projects

## 🛠️ How to Use This Repo

- **JS files** (e.g. inside `basics/`, `basics2/`, `control_flow/`, `iterations/`, `oops/`) can be run directly with Node.js:
  ```bash
  node basics/01_variables.js
  ```
- **HTML files** (e.g. inside `dom/`, `events/`, `projects/`) should be opened in a browser to see them in action. Just double-click the `index.html` file or use a tool like the VS Code "Live Server" extension.

## ✅ How I'm Using This Repo to Learn

1. Every new concept gets its own file/folder, named clearly after the topic.
2. Once a topic is done, it's added to the **Topics Covered** list above.
3. Small projects in `projects/` are built to apply multiple concepts together.
4. This README is updated as new topics/projects are added, so it always stays an up-to-date map of everything learned.
